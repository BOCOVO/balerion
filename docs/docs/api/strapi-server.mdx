# Balerion Server

:::info

Current state: **Stable**

:::

The Balerion server module permits to generate a Balerion http server.

## Module: Balerion server

### `createServer(balerion)`

- `balerion`: [<Type>Balerion</Type>](Balerion)
- Returns: <Type>BalerionServer</Type>

```javascript
const server = createServer(balerion);

server.listRoutes();
```

### `BalerionServer.app`

- [<Type>KoaJS</Type>](https://devdocs.io/koa/index)

Balerion projects are using KoaJS to run the NodeJS server.

### `BalerionServer.router`

- [<Type>@koa/router</Type>](https://github.com/ZijianHe/koa-router#router-)

Balerion projects are using a dependency of KoaJS called @koa/router.

### `BalerionServer.httpServer`

- [<Type>http.Server</Type>](https://nodejs.org/docs/latest-v18.x/api/http.html)

The Balerion's HTTP server.

### `BalerionServer.api(name)`

- `name`: <Type>String</Type>
- Returns: [<Type>BalerionAPIs</Type>](#balerionapis)

Getter for apis available in Balerion

### `BalerionServer.use(...args)`

- [<Type>KoaApp.use</Type>](https://devdocs.io/koa/index#appusefunction)

Shortcut for Koa `app.use(...args)` method.
